Programming

In Programming we have learned the programming language of Java, at the moment we are getting eased into the language by creating shapes and learning about procedures and different variables so that they can be adjusted in size and looped into different patterns, we have then used this knowledge to create mini games that the user can interact with using the keys, as you can see in the first screen shot we had developed a pong game using delta commands so the user can move the bat up and down and had given the ball code so that it can bounce against the wall and back towards the bat. And in the second game we have gone with the same prinsibles but instead have added a collition code where the rocket known as the defender once it makes contact with the defender will then stop and the user will have to start again in order to continue.

Computer Systems Fundamentals
